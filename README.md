# Pembuatan-Agen-AI-Chatbot-Web-dengan-n8n-dan-Gemini-1
Proyek ini membuat AI chatbot web menggunakan n8n dan Google Gemini. Chatbot berfungsi sebagai asisten virtual untuk memberikan informasi mengenai Bawaslu Kota Depok, seperti tugas, kegiatan pengawasan pemilu, berita, dan layanan masyarakat melalui percakapan di web.
